Lab One

1. Personal Webpage

This page is created from scratch using HTML, CSS and basic JavaScript. My editor of choice is Brackets with the extensions Emmet, autosave every edit!, beautify, and the built-in live preview. I use XAMPP in creating a local web server for website testing and deployment purposes, GIMP 2 for image editing, FileZilla to upload files to the server by FTP, Google Docs in embedding documents, and twitter for embedded tweets, forecast.io widget for weather, econmic times widget for the tech news, and Google Maps API for my maps. My references are www.w3schools.com, www.developer.mozilla.org, developers.google.com and www.stackoverflow.com.

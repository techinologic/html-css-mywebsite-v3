Projects I’m Currently Working on:

Senior Project - Object Detection AI with Facial Recognition

We are working on a computer vision and image processing AI that will detect objects of certain class such as cars, smartphones, or a ball.
We will also incorporate a facial recognition system capable of identifying a face of a person in live videos. We will use the skills and techniques we acquired from our AI course and apply our newly learned machine learning principles to complete the project.


An object detection software without face recognition. Photo from Intel.

Counting Algorithm

I'm developing an algorithm that will efficiently count objects, represented by integers in a huge matrix. For a start, I managed to count the circles in the image below but it needs to be done prior to generating the image.


Circles counted and traced using a Hough Circle Transform Algorithm in MATLAB.

App Development

I’m also working on a mobile app for Type 1 diabetics (being one) that will make our insulin dose calculation easier. It scans the barcode of a food then retrieves the nutrition information using an API and calculates the dose needed. I'm planning to integrated this software to my insulin pump to automate the whole process. The Android version is at its final stage of development so stay tuned.

About This Page

This page is created from scratch using HTML and CSS. Currently the 3rd version, this time showcasing the CSS Grid Layout with built in responsiveness with the help of CSS flexbox.
My editor of choice is Brackets with the extensions Emmet, autosave every edit! and beautify. I use XAMPP in creating a local web server for website testing and deployment purposes, GIMP 2 for image editing, FileZilla to upload files to the server by FTP, Google Docs in embedding documents, twitter for embedded tweets, forecast.io widget for weather, econmic times widget for the tech news, and Google Maps API for my maps. My web references are w3schools, Mozilla Developers, Google Developers, CSS-Tricks, Stackoverflow and materials learned from the class.
All photos presented here are either my work or taken from www.pexels.com which are licensed under the Creative Commons Zero license (CC0). This means that they are free for any personal and commercial purpose. Learn more about the creative commons license here.

By the end of this course, I expect to:

Learn the structure and functionality of the world wide web, create dynamic web pages using a combination of HTML, CSS, and JavaScript, apply essential programming language concepts, use different web technologies and frameworks, develop web applications and publish my own fully functional webpages for the world to see. I expect to develop a working model for creating my own personal or business websites in the future and be fully prepared to take the next step in a more advanced web development or design course or specialization.

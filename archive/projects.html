1. Personal Webpage

This page is created using the following html tags:

Tag Description
<!DOCTYPE>  Defines the document type
<html>  Defines an HTML document
<head>  Defines information about the document
<title>  Defines a title for the document
<li>  Defines a list item
<a>  Defines a hyperlink, which is used to link from one page to another
<body>  Defines the document's body
<ul>  Defines an unordered list
<h1> to <h3> Defines HTML headings
<p>  Defines a paragraph
<br>  Inserts a single line break
<em>  Show as emphasized text
<strong>  Defines important text
<th>  Defines a header cell in a table
<td>  Defines a cell in a table
<tr>  Defines a row in an HTML table.
<br>  Inserts a single line break
<img>  Defines an image
<iframe>  Defines an inline frame
<header>  Defines a header for a document or section
<link>  Defines the relationship between a document and an external resource (most used to link to style sheets)
<script>  Defines client-side script (JavaScript)
<div>  Defines a section in a document
<nav>  Defines navigation links
<blockquote>  Defines a section that is quoted from another source
<footer>  Defines a footer for a document or section
<!--...--> Defines a comment
Source: Descriptions taken from W3Schools's HTML element reference.

My editor of choice is Brackets with the extensions Emmet, autosave every edit! and indentator. I use FileZilla to upload files to the server using FTP.
